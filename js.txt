Sobre o JavaScript

    Declaração de Variável
        const modal = x
            - const: declara uma variável;
            - modal: nome da variável;
            - =: operador de atribuição, estando atribuindo um valor para essa variável;
            - x: valor a ser atribuído;
    
    Funções
        function nomeFunção(){}
            - function: palavra-chave para declarar uma função;
            - nomeFunção: nome da função que voce deseja;
            - (): onde passa os parãmetros, se tiver vazio, significa que a função não tem nenhum parâmeto;
            - {}: dentro colocar o que a função vai fazer;
    
    DOM - interações com o HTML
        - document.querySelector('seletor') 
            - document: documento HTML inteiro carregado na página;
            - .: acesso a uma propriedáde ou método do objeto 'document';
            - querySelector: nome do método(função) que faz a busca;
            - (): chama a função dentro dele('seletor') coloca os dados que o querySelector deve buscar;
    
    Programação Assíncrona
        - async: Define uma função assíncrona. Permitindo usar await dentro dela para esperar o resulado de opeações como fetch()
    
    Criação de Objetos / Classes
        - new URLSearchParams(): cria um objeto que permite ler, editar, adicionar ou remover os parâmetros de uma URL;
